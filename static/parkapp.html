역할

팀에서 안드로이드 코어 개발자로써 중요 기능들을 구현

외부 아이디로 로그인

카카오SDK, 네이버SDK, 페이스북SDK를 이용해서 외부 아이디로도 서비스를 이용 할 수 있게 함.

GCM, FCM을 이용한 푸쉬알림

사용자가 앱을 사용 중이지 않을 때도 알림이 있으면 GCM서버를 이용해서 기기에 푸쉬 알림을 보냄

KMC 휴대폰 인증

사용자의 휴대폰 인증을 위해 KMC(한국 모바일 인증)이 제공하는 파일을 이용해서 휴대폰 인증 서비스 제공
